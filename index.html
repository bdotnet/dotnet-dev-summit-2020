About The Conference

.NET Dev Summit 2020 is a free, virtual developer conference co-organized by the Bangalore DotNet UG (BDotNet), XHackers Xamarin community and Microsoft. Come and Join us to learn about the latest and greatest of Microsoft Technologies directly from Microsoft folks, Microsoft MVPs, Industry experts and Community rockstars.

Schedule

  • Aug 29

  • 09:45 AM

    Welcome/Event Briefing

  • 10:00 AM - 11:00 AM

    What’s New & Next for Xamarin Developers

    By James Montemagno
    Join James Montemagno, Principal Lead PM for .NET Community, and he shows off the latest and greats for Xamarin developers. This includes great enhancements to the core platforms, cross-platform development with Xamarin.Forms, and awesome new tooling to be more productive than ever. He will then talk about the future with .NET MAUI and what this next evolution of Xamarin.Forms bring for developers and why he is so excited.

  • 11:00 AM - 11:15 AM

    Break/Networking

  • 11:15 AM - 12:15 PM

    Full Stack Web Development with .NET Blazor

    By S Ravi Kumar
    Today, nearly all browser-based apps are written in JavaScript but there’s no good reason to limit our industry to basically one language. Blazor is a new web UI framework from the ASP.NET team that aims to bring .NET applications into all browsers (including mobile) via WebAssembly. In this talk, Join Ravi (aka TechieRathore) where he talks about: 1) Confusion about choosing Web UI framework 2) What is Web Assembly, Blazor and How it works 3) Future of Web Assembly & Blazor 4) Creating Web & Desktop applications using Blazor along with a cool demo.

  • 12:15 PM - 01.15 PM

    Microservices in .NET

    By Nish Anil
    Microservices can be hard; but, Hear directly from Nish Anil, Sr. Program Manager at Microsoft about why microservices are important and why you should care about them. Learn about new tooling the .NET team is working on to help you build microservices faster. In this talk, he will talk about .NET latest updates, Project Tye, VS Tooling, creating new Microservice from scratch, Deploy to Docker and K8s. We bet you don't want to miss this one.

  • 01:15 PM - 02:00 PM

    Lunch

  • 02:00 PM - 03:00 PM

    Let us DAPR!

    By Amresh Krishnamurthy
    DAPR- Distributed Application Runtime is an event-driven, portable runtime for building microservices on cloud and edge. Join Amresh Krishnamurthy (Technical Architect at JCPenney India) where he will talk about the building blocks of DAPR and how DAPR enables us to solve some of the complexities in writing distributed cloud-native applications.

  • 03:00 PM - 03:45 PM

    Ask Me Anything with Experts/MVPs

  • 03:45 PM - 04:00 PM

    Wrap up
